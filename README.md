# AI Quantum Learning Lab

Browser-first learning environment for exploring how AI and quantum computing can work together in finance.

## Goals
- Keep every step visible and explainable
- Use local Qiskit simulation only in the first phase
- Combine business logic, classical methods, and quantum demonstrations
- Use AI as a guide layer, not as a black box

## Stack
- GitHub for code and documentation
- GitHub Pages for docs/demo layer
- Google Colab for browser-based Python execution
- OpenAI API for the guide layer
- Qiskit local simulation for quantum experimentation

## First demos
1. Credit Portfolio Monte Carlo
2. Credit Selection Optimization

## Quick start
1. Create a GitHub repo from this project
2. Open `notebooks/00_setup_and_sanity_check.ipynb`
3. Then run `notebooks/01_credit_portfolio_monte_carlo.ipynb`

## Notes
- Replace `YOUR_USERNAME` in notebooks with your GitHub username
- Set `OPENAI_API_KEY` before using the guide layer
