# AI + Quantum Learning Lab

This project is a browser-first learning environment for exploring how AI and quantum computing can work together in finance.

## Goals
- keep every step visible
- use local Qiskit simulation only
- combine business logic, classical methods, and quantum demonstrations
- use AI as a guide layer, not as a black box

## Demos
- Credit Portfolio Monte Carlo
- Credit Selection Optimization

## Components
- GitHub Pages for visible docs/demo
- Colab for Python and Qiskit execution
- OpenAI API for the guide layer
- Qiskit local simulation for quantum experimentation
